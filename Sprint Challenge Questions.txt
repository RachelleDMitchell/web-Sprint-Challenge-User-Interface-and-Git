1. If you were to describe semantic HTML to the next cohort of students, what would you say?

    I would say that semantic HTML is HTML attributes and tags that convey meaning.


2. Name two big differences between display: block; and display: inline;.

    display:block puts the content on a whole new line where display:inline puts the content on the same line as the rest of the content.


3. What are the 4 areas of the box model?

    The 4 areas are the content edge, padding edge, border edge and margin edge.


4. While using flexbox, what axis does the following property work on: align-items: center?

    That would be the cross axis.


5. Explain why git is valuable to a team of developers.

    Git is valuable to developer teams because it lets multiple people edit and upload versions of a main product without messing up the original version. It also lets you go back to previous versions if needed.
